# 画像フォルダ構成

各セクション用の画像を格納するフォルダです。

## 📁 フォルダ構成

### 01_header - ヘッダー
- ロゴ画像
- ヒーロー画像

### 02_past-events - これまでのフラコレ
- 過去のイベント写真
- 展示作品画像

### 03_schedule - スケジュール
- （必要に応じて）

### 04_venue - 開催場所
- 会場外観
- 会場内部
- アクセス地図

### 05_goods - グッズ販売
- グッズ商品画像

### 06_members - 出展メンバー
- メンバープロフィール画像
- 作品サンプル

### 07_artwork-sales - 作品販売
- 販売作品画像

### 08_supporters - 支援者の皆様
- （必要に応じて）

### 09_support-goods - フラコレ応援グッズ
- 応援グッズ画像

### 10_staff - 運営メンバー
- スタッフプロフィール画像

## 💡 使用方法

1. 対応するフォルダに画像ファイルを配置
2. TXTファイルで画像ファイル名を指定
3. Codex に更新反映を依頼（または手動で反映）
