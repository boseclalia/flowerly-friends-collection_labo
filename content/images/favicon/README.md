# ファビコン・アイコン画像フォルダ

## 必要な画像とサイズ

### 基本ファビコン
- **favicon.ico** - 16×16, 32×32, 48×48px（マルチサイズ.icoファイル）
- **favicon-16x16.png** - 16×16px
- **favicon-32x32.png** - 32×32px

### Apple Touch Icon（iPhone/iPad）
- **apple-touch-icon.png** - 180×180px（推奨）
- **apple-touch-icon-152x152.png** - 152×152px（iPad用）
- **apple-touch-icon-120x120.png** - 120×120px（iPhone用）

### Android/Chrome用
- **android-chrome-192x192.png** - 192×192px
- **android-chrome-512x512.png** - 512×512px

## デザイン要件
- **基本デザイン**: Flowerly Friends Collectionのロゴまたは花のモチーフ
- **背景**: 透明または白背景
- **色**: ブランドカラー（ピンク系グラデーション）
- **視認性**: 小さいサイズでも認識できるシンプルなデザイン

## ファイル形式
- .ico（Windows用マルチサイズ）
- .png（透明背景対応、高品質）
- 全て正方形で作成

## 推奨ツール
- オンライン: https://realfavicongenerator.net/
- 一つの高解像度画像（512×512px以上）から全サイズ生成可能
